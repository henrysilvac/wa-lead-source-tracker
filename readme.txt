Starter kit MVP para capturar UTMs y añadirlos al mensaje de WhatsApp.

== Description ==

Este plugin captura parámetros UTM y click IDs desde la URL, los guarda en localStorage y actualiza enlaces de WhatsApp con un mensaje dinámico.

== Installation ==

1. Sube el zip del plugin desde Plugins > Añadir nuevo > Subir plugin.
2. Activa el plugin.
3. Ve a Ajustes > WA Lead Source Tracker.
4. Configura número, selector y plantilla de mensaje.
5. Usa el shortcode [wa_lead_button] o aplica la clase CSS configurada a botones existentes.
